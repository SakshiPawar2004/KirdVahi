Initialize Schools in Firebase

⚠️ Security Notice: Before using this tool, you MUST replace the placeholder Firebase configuration with your own credentials.

Steps:
  1. Go to Firebase Console
  2. Select your project → Project Settings → Your apps → Web app
  3. Copy your Firebase configuration
  4. Replace the placeholder values in the firebaseConfig object above
  5. Make sure your Firestore security rules allow writes
⚠️ Never commit your actual API keys to Git!

This script will create 6 schools in your Firebase Firestore database:

  • School 1 (admin1 / School1@2024)
  • School 2 (admin2 / School2@2024)
  • School 3 (admin3 / School3@2024)
  • School 4 (admin4 / School4@2024)
  • School 5 (admin5 / School5@2024)
